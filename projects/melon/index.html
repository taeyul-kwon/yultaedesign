Melon은 한국을 대표하는 음원 스트리밍 플랫폼으로, 사용자의 취향을 기반으로 다양한 음악 경험을 제공하는 서비스입니다.

이번 프로젝트에서는 취향 중심의 홈 구성, 트렌드 확인 탭, 음악 방송, 맞춤 태그 추천, 나만의 음악 서랍 등 Melon의 주요 기능을 쉽고 직관적으로 전달하는 2D 모션 그래픽 영상을 제작했습니다.

해당 영상은 Melon이 단순한 음악 재생 서비스를 넘어, 취향 기반의 음악 탐색과 개인화된 경험을 제공하는 플랫폼임을 효과적으로 보여주는 데 목적을 두었습니다.
